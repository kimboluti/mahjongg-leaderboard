How Points Are Tracked

Winner earns the hand's point value plus any bonuses — these accumulate over the season.
Other 3 players each receive a loss (tracked for win-rate, no point deduction).
Wall game: no points, no wins — all 4 players get a wall-game tally.
Absent members are not counted — only log the 4 who actually played.

Schedule

Meets on the 1st and 3rd Wednesday of each month at the NLNA Community Center, 700 N 3rd Street, Philadelphia PA 19123. With 12 members, up to 3 tables of 4 can play each session.
About This Leaderboard

How It Works

This leaderboard reads live from a Google Sheet that is updated after each session. The next time you load or refresh the page, you'll see the latest standings. No app to install — just bookmark this page!

📋 Access to the score sheet is limited. If you notice a missing result, a scoring error, or have any questions about the leaderboard, contact Ellie.

What the Columns Mean

Each player's row tracks the following across all games played this season:

Points — total points earned from winning hands (accumulates all season)
Wins — number of games won (Mah Jongg declared)
Losses — number of games where someone else at your table won
Walls — number of wall games (no winner) you were present for
Games — total games played (wins + losses + walls)
Win Rate — percentage of your games that you won
Avg/Win — your average point value per winning hand

Click any column header to re-sort. Click a player's row to expand their individual stats.

How Winning Hands Are Recorded

Each winning game includes a note identifying the hand played, using the format from the NMJL card:

Format: CATEGORY_ROW

13579_1   → "2024 Hands" section, row 1
NEWS_5    → "NEWS" section, row 5
CONSEC_3 → "Consecutive Run" section, row 3

This follows the naming convention of the annual NMJL card, where each section has a category name and numbered rows. You can see the winning hand in the Game History tab.

Refreshing the Page

After Ellie updates the score sheet following a session, click the ↻ Refresh button on the Leaderboard tab to pull in the latest results. The page does not update automatically — a manual refresh is needed.
